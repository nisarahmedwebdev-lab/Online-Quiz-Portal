JSON Format Examples
Single Quiz Format (Recommended):
{
  "quiz_title": "Technical and Business Writing",
  "quiz_description": "30 important multiple-choice questions",
  "questions": [
    {
      "question_text": "What is the main goal of technical writing?",
      "marks": 1,
      "options": ["To entertain", "To inform clearly", "Express opinions", "Tell stories"],
      "correct_option": 2
    }
  ]
}
Multiple Quizzes Format:
[
  {
    "quiz_title": "Quiz 1",
    "questions": [
      {
        "question_text": "Question 1?",
        "options": ["A", "B", "C", "D"],
        "correct_option": 1
      }
    ]
  },
  {
    "quiz_title": "Quiz 2",
    "questions": [
      {
        "question_text": "Question 2?",
        "options": ["A", "B", "C", "D"],
        "correct_option": 2
      }
    ]
  }
]
Note: Your JSON data should follow one of these formats. The system now supports both single quiz objects and arrays of multiple quizzes.
